# ellocharlie

**CRM Built for Startups** — One connected system for your entire customer journey.

> Building a $100M agent-driven company with 5 AI agents and world-class engineering discipline.

## The Product

ellocharlie connects your CRM, helpdesk, onboarding, docs portal, and status page into one system. No more scattered context. No more churn from broken handoffs.

**Core** ($45/seat/mo) — CRM & Sales, Customer & Project Management, Live Chat & Helpdesk, Ticket Tracker, Status Page

**Plus** ($70/seat/mo) — All Core + Custom Domains, Onboarding Call, Beta Features, Priority Support
